Ubuntu安裝java8

Edward Elric, 
May 23rd, 2015 · 1 min read
  1. 添加 ppa
1sudo add-apt-repository ppa:webupd8team/java
  1. 更新系統
1sudo apt-get update
  1. 安裝 java8
1sudo apt-get install oracle-java8-installer
  1. 檢測是否成功
1java -version

結果如下

1java version "1.8.0_45"
2Java(TM) SE Runtime Environment (build 1.8.0_45-b14)
3Java HotSpot(TM) 64-Bit Server VM (build 25.45-b02, mixed mode)
  1. 設置環境變量
1sudo apt-get install oracle-java8-set-default
  1. 版本切換
1sudo update-java-alternatives -s java-8-oracle
