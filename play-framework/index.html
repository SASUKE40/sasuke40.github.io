Play Framework項目初始化

2015-05-23

安裝 Activator

先下載Activator 解壓到D:\Program Files\activator-1.3.2 將D:\Program Files\activator-1.3.2加到環境變量 Path 中 在你的項目目錄中執行以下命令，驗證是否安裝成功

activator -help

創建 Play 項目

activator new demo play-java
cd demo
activator

安裝 Eclipse 插件

Scala IDE 在 Eclipse 中 Install New Software 增加宰割 Update Site，記得安裝 Play Support

創建 Eclipse 項目

eclipse with-source=true

跑起來看看

run

在瀏覽器中打開http://localhost:9000，項目就跑起來了 ╰(￣ ▽ ￣)╮
