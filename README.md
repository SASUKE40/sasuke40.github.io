# Edward Elric's Blog

This is my own blog, using `Gatsby` to pre render.
And all post are under `/content/posts` folder

## Usage

### Looking for Something?

- `/content` all blogs

- `/src` source code

### Test and Build Blog

```bash
npm i           # install all dependencies

npm run dev     # dev mode

npm run build   # build blog
```
