# Edward Elric's Blog

This is my own blog, using `Gatsby` to pre render.
And all post are under `/content/blog` folder

## Usage

### Looking for Something?

- `/content` all blogs

- `/src` source code

### Test and Build Blog

```bash
yarn            # install all dependencies

yarn develop    # dev mode

yarn deploy     # deploy blog      
```
